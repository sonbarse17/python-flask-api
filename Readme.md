# Python Flask API

Flask API server with modern web interface.

## Features
- Flask framework
- API endpoints
- Modern UI
- Docker ready

## Quick Start
```bash
docker build -t python-flask-api .
docker run -p 9001:9001 python-flask-api
```

Visit: http://localhost:9001
